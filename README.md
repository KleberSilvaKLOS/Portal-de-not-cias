🌐 Portal de Notícias TechNews
Este projeto consiste em um front-end para um portal de notícias especializado em tecnologia emergente. A interface foi construída para organizar grandes volumes de informação de forma hierárquica e visualmente atraente, utilizando as melhores práticas de semântica e layout.

🛠️ Tecnologias e Recursos
HTML5 Semântico: Estruturação utilizando tags avançadas (main, section, article, aside, figure) para garantir melhor indexação em motores de busca (SEO) e acessibilidade.

CSS Grid Layout: Utilização intensiva de display: grid e grid-flow-col para criar um design complexo, mas flexível e organizado.

Tipografia: Integração com Google Fonts (família Archivo) para garantir legibilidade e uma estética moderna.

Gestão de Assets: Organização sistemática de ícones em SVG e imagens otimizadas para carregamento rápido.

🏗️ Arquitetura do Layout
O portal está dividido em seções estratégicas que facilitam a navegação do usuário:

Header Dinâmico: Possui dois níveis de navegação:

Primário: Acesso ao menu, logo central e ferramenta de busca.

Secundário: Filtros por categorias (IA, Blockchain, Realidade Aumentada, etc.).

Featured (Destaques): Uma grade principal que combina uma notícia de grande impacto com quatro sub-notícias em formato de card.

Weekly (Mais lidas): Seção vertical focada nas notícias com maior engajamento da semana.

AI Section: Área dedicada a artigos longos sobre Inteligência Artificial, unindo títulos impactantes, descrição e imagens laterais.

Aside (Barra Lateral): Espaço para anúncios (ads) e uma seção de "Veja mais" para manter o fluxo de leitura do usuário.

✨ Funcionalidades Destacadas
Componentização via Classes: Uso de classes utilitárias como container, grid e gap-xx para manter a consistência do espaçamento.

Content-Tags: Sistema de etiquetas coloridas para identificar rapidamente o tema de cada notícia (ex: Robótica, Software, Drones).

Interface Limpa: Foco no conteúdo visual e textual, utilizando figcaption para legendas contextuais dentro dos cards de notícias.

📂 Como Executar o Projeto
Clone este repositório para sua máquina local.

Certifique-se de manter a estrutura de pastas:

/assets/ (Ícones e imagens)

/styles/ (Arquivos CSS)

Abra o arquivo index.html em um navegador.
